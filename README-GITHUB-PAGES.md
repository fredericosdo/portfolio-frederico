# Portfólio Online - GitHub Pages

Este projeto foi preparado para publicação no **GitHub Pages**, conforme solicitado na atividade prática.

## Arquivos principais
- `index.html`
- `style.css`
- `script.js`
- `assets/`

## Como publicar no GitHub Pages
1. Crie um repositório público no GitHub.
2. Envie todos os arquivos desta pasta para o repositório.
3. No GitHub, abra **Settings > Pages**.
4. Em **Build and deployment**, selecione:
   - **Source:** Deploy from a branch
   - **Branch:** `main` (ou `master`)
   - **Folder:** `/root`
5. Salve e aguarde alguns minutos.
6. O GitHub exibirá o link publicado do site.

## Sugestão de nome do repositório
Para facilitar a avaliação, use um nome com seu nome:
- `frederico-oliveira-portfolio`
- `portfolio-frederico`
- `curriculo-online-frederico`
